## volatile이란
- `volatile` 키워드는 자바 변수를 Main memory에 저장하겠다고 명시하는 것
- 매번 변수의 값을 read할 때마다 CPU 캐시에 저장된 값이 아닌 Main memory에서 읽는 것
- 매번 변수의 값을 write할 때마다 Main memory에 저장하겠다는 것(캐시 사용X)
